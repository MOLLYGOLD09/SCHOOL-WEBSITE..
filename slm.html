How to Use the SLMs

Step-by-step guide for students and parents

1️⃣ Select Module Type

Choose the appropriate module type from the options above (JHS, SHS, MATATAG, etc.)

2️⃣ Click Drive Link

Click on the Google Drive link to access the folder containing the modules

3️⃣ Navigate Folders

Browse through the folders organized by grade level, subject, and quarter

4️⃣ Download or View

Download the PDF modules or view them directly in Google Drive

Important Notices

Please read before using the SLMs

⚠️ File Format

All modules are in PDF format. You need a PDF reader to view them. Download Adobe Acrobat Reader for free.

📱 Mobile Access

Modules can be viewed on mobile devices. For best experience, use a tablet or computer.

🔄 Updates

Modules are regularly updated. Check back for new versions and additional content.

📞 Support

If you encounter issues with downloading or viewing modules, contact the LRMS office.
